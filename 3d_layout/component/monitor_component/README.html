3D教学/学习组件，提供可视化3D实验环境。

在网页中的使用方法

添加如下引用，即可。 <script src="<组件的路径>/component/web3D.js" type="text/javascript" charset="utf-8"></script> 或简写为： <script src="<组件的路径>/component/web3D.js"></script>

快捷键


l (location) ：设置可选择的3D对象，当前被选中的3D变换元素会切换到新选择器指明的第一个元素。具体操作：输入CSS选择器
x,y,z切换变换的坐标轴
p (party)：切换到同时按照x、y轴进行变换
t（transform）：进行平移变换
r（rotate）：进行旋转变换
s（scale）：进行缩放变换
k（keep）或者鼠标双击本元素 ： 保存当前变换，后续将在保存的变换的基础上进行变换。
a (axis) ：隐藏/显示坐标轴
g (animation) ：播放一次旋转动画
n (none)：隐藏/显示当前选定的元素
h (help) ：显示此帮助
c (command)：查看当前3D变换的transform的变换命令
e (edit)：设置任意一个CSS属性值，请输入用竖线（|）作为分隔的2个参数（”css属性名 css属性值“或"选择器，json格式的CSS属性集合"）或三个参数（选择器，css属性名，css属性值）
鼠标双击某元素，切换新的3D元素

JavaScript API调用格式

$watch(selectorMain, transformObject, axis, action, objectSets) 监视某3D对象，被监视的对象可以用鼠标对其进行可视化3D变换


selectorMain参数是用于捕获鼠标坐标的元素，CSS选择器；
transformObject是被控制变换的3D元素，CSS选择器;
axis参数，设定变换的坐标轴。可以取如下四个值：'x'、'y'、'z'、'xy',默认'xy'。
action参数，设置变换种类：'rotate'表示旋转，'translate'表示平移 ，'scale'表示伸缩。默认’scale'。
第五个参数，objectSets是鼠标双击的时候，可以切换过去的3D变换元素的选择器

注意事项


进行3D变换的时候，objectSets参数中，请尽量避免出现class属性值完全一致的情况，否则属性一致的元素会混淆“上一轮”的3D变换。
被选择作为3D变换的对象的class值，必须是唯一的，不允许与其它元素的class值相等，但可以相交。

$init3D(selectorMe, transform) 3D对象初始化


selectorMe,被初始化对象的css选择器
transform,transform变换的值，3D命令集合

$cube(selectorMe,lenght,width,height,imgs,opacity, contextLocation，faces)



参数selectorMe是一个选择器，长方体将在此元素内部构建
参数lenght是立方体得长度
参数width是立方体得宽度
参数height是立方体的高度
参数imgs是一个数组，里面放着0~6个图像（顺序为 上、下、前、后、右、左）文件的路径，路径是以”/”开头的绝对路径或者以设定的路径开头 ，填写空字符串或错误文件名，将缺失对应的面。
参数opacity是各个面的透明度
参数contextLocation是内容展现的那个面，取值0（放在原本的位置）或1~6（上、下、前、后、右、左）
参数faces是一个数组，里面包含1~6个css选择器或jquery对象.若提供此参数，则cube的6个面，由此参数决定。

$cylinder(selectorMe, r, height, color, img, level,scaleModeNum) 创建空心的圆柱


selectorMe ,css选择器，在此元素上构建圆柱
r 圆柱的半径
height 圆柱的高度
color 圆柱的颜色
img 圆柱的表面贴图
lever 圆柱的扇面数（3~N）
scaleModeNum 贴图的伸展模式（1---level个贴图组成，2--- 少量的数个贴图，3---唯一一个贴图伸展铺满柱面），默认3 。

$line(selectorMe, point1, point2, color, width, unit) 画空间直线



画3D空间中的线段.
selectorMe 线段所在的容器，CSS选择器或JQ对象
point1 起始点
point2 结束点
color 线段的颜色
width 线段的宽度
unit 坐标的刻度单位，省略此参数，默认为"px"
例如 $line(".div002",[2,3,10],[3,1,5],"#ff0000",2,"px")

$sphere(selectorMe, r, color, img, levelW, levelH, util) 创建空心球体


selectorMe 用于建立球体的容器，CSS选择器或JQ对象
r 半径
color 颜色
img 表面贴图
levelW 球瓣的横向密度，正整数，建议设置在1~30以内
levelH 球瓣的竖向密度，正整数，建议设置在1~30以内
util 度量单位，默认"px"

$thickness(selector, px) 给容器元素（典型是div元素）增加厚度


selector是选择器
px是厚度（单位px）

format(...) 扩展js字符串的新方法，实现占位符替换


例如：
"rotateX({0}deg) rotateY({1}{2})".format(30,"25","rad") ,结果返回字符串"rotateX(30deg) rotateY(25rad)"
